# Domain Model 章節說明

本目錄包含 TDD 文件的 Domain Model 章節，定義所有 Entity 和 Value Object。

## 文件列表

- `01_domain_model.md` - Domain Model 定義
- `02_domain_model_uml_standard.md` - Domain Model UML 標準關係圖

## 說明

Domain Model 章節包含：

- 所有 Entity（實體）定義
- 所有 Value Object（值物件）定義
- Entity 和 Value Object 的關係圖
- 語意化關係說明

## Domain Model 類型

1. **Entity（實體）**：有唯一識別碼的業務物件
2. **Value Object（值物件）**：沒有唯一識別碼，透過值比較
