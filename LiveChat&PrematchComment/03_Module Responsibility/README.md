# Module Responsibility 章節說明

本目錄包含 TDD 文件的 Module Responsibility 章節，列出並說明所有模組的職責。

## 文件列表

- `01_module_responsibility.md` - 模組職責說明

## 說明

Module Responsibility 章節包含：

- Feature Modules（Thin Reducer）
- UseCase Modules
- Repository Modules
- Client Modules
- API Modules
- Shared Modules（如適用）

所有說明使用表格呈現，以便快速查找與比較。
