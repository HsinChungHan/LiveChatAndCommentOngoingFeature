# Integrated Service-Level Sequence Diagram 章節說明

本目錄包含 TDD 文件的 Integrated Service-Level Sequence Diagram（ISSD）章節，提供跨場景、跨 Feature、跨 UseCase 的整合序列圖。

## 文件列表

- `01_full_integration_flow.md` - 完整整合流程序列圖

## 說明

Integrated Service-Level Sequence Diagram 用於呈現整體系統的 high-level 協作，顯示：

- Shared Modules（AuthFeature、NicknameFeature、UserInfoRepository 等）
- Repository、Client、API 如何共同運作
- 跨 Feature 的互動流程
- 外部 Package 的整合方式
