# UseCase Input & Output Model

本目錄包含所有 UseCase 的 Input 和 Output Model 定義。

## 文件列表

- `01_usecase_input_output.md` - 所有 UseCase 的 Input 和 Output Model 定義

## 說明

本章節定義了所有 UseCase 的 Input 和 Output Model，包括：

- **Input Model**：Feature 呼叫 UseCase 的參數封裝
- **Output Model**：UseCase 回傳給 Feature 的 Domain 結果

所有 Input/Output Model 必須使用 Domain Model，不得使用 DTO。
