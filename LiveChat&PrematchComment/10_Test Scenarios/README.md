# Test Scenarios

本目錄包含所有 UseCase 的測試場景定義。

## 文件列表

- `01_test_scenarios.md` - 所有 UseCase 的測試場景定義

## 說明

本章節定義了所有 UseCase 的測試場景，包括：

- Basic Flow（正常成功流程）
- Branch Flow（分支行為）
- Optional Flow（可選流程）
- Loop Flow（循環流程）
- Error Flow（錯誤流程）
- Recover Flow（恢復流程）
- Edge Case（邊界情況）

測試場景由序列圖中的 alt/opt/loop 自動推導。
