# Review Cursor Conversations

統整與 Cursor 的對話，提取重要決策和學習要點。

## 使用方式

直接在 Cursor 中執行此命令，或說「統整對話」

## 功能

- 統整今天與 Cursor 的重要對話
- 提取重要決策
- 提取學習要點
- 生成對話摘要

## 執行

請協助我統整今天與你的對話：

1. 回顧今天的重要對話（如果有的話）
2. 提取重要決策：
   - 實作決策
   - 架構決策
   - 技術選型
3. 提取學習要點：
   - 新學到的知識
   - 最佳實踐
   - 改進建議
4. 生成對話摘要
5. 更新工作日誌中的「與 Cursor 的對話統整」部分

## 輸出格式

### 重要決策
- [決策 1]: [描述]
- [決策 2]: [描述]

### 學習要點
- [要點 1]: [描述]
- [要點 2]: [描述]

### 改進建議
- [建議 1]: [描述]
- [建議 2]: [描述]
