# Summarize Daily Work

統整當天的工作，包含與 Cursor 的對話統整。

## 使用方式

直接在 Cursor 中執行此命令，或說「統整今天的工作」

## 功能

- 讀取當天的工作計劃和工作日誌
- 生成工作統整文件
- 包含與 Cursor 對話統整模板

## 執行

```bash
cd /Users/reedhsin/Documents/codebase/fcom-iOS/TDDs/LiveChat\&PrematchComment && ./scripts/summarize_daily_work.sh
```

## 生成的文件

- `16_Cursor_Workflow/daily_summaries/YYYY-MM-DD.md`

## 後續步驟

執行後，請手動填寫：
1. 完成項目（從工作日誌中提取）
2. 與 Cursor 的對話統整
3. 工作統計
4. 反思與學習
