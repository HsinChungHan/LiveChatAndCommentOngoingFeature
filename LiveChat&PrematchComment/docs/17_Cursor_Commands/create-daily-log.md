# Create Daily Log

建立今天的工作日誌文件。

## 使用方式

直接在 Cursor 中執行此命令，或說「建立今天的工作日誌」

## 功能

- 根據今天的日期建立工作日誌文件
- 使用預設模板，包含與 Cursor 對話統整部分

## 執行

```bash
cd /Users/reedhsin/Documents/codebase/fcom-iOS/TDDs/LiveChat\&PrematchComment && ./scripts/create_daily_log.sh
```

## 生成的文件

- `15_Daily_Logs/YYYY-MM-DD.md`
