# Update Implementation Status

更新實作狀態追蹤表，檢查實作與 TDD 的一致性。

## 使用方式

直接在 Cursor 中執行此命令，或說「更新實作狀態」

## 功能

- 檢查今天的實作進度
- 更新 implementation_status.md
- 檢查實作檔案是否存在
- 檢查實作是否符合 TDD 規範

## 執行

請協助我：
1. 讀取 `13_Implementation_Status/implementation_status.md`
2. 根據今天的 git commits 找出相關的 tickets
3. 檢查實作檔案是否存在
4. 更新實作狀態（進行中/已完成）
5. 檢查實作是否符合 TDD 規範
6. 更新 implementation_status.md
